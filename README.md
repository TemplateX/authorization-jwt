# authorization-jwt
JWT-система аутентификации на FastAPI с разделением ролей (Admin/User/Moderator/Editor). Реализованы Access/Refresh токены, асинхронная SQLAlchemy, SQLite, Pydantic. Защищенные роутеры (Depends), Logout, смену пароля, эндпоинты для ролей, просмотр, редактирование, мягкое удаление профиля. Автоматическая инициализацию бд при старте.
